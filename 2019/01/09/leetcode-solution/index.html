1.Two Sum

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

Given nums = [2, 7, 4, 5], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

题解：

如果我们有一个字典,它的key是[2, 7, 4, 5]，它的value包含了它在序列中的位置和它所要找到那个数。则通过这个字典可以观察到整个序列中相加为9的所有组合。若这个序列比较长，则可能出现多个相加为9的组合。这样的方法虽然可以找出你要的所有组合，但是做这样的一个字典本身需要n次计算，而搜索又需要(n-1)*n次。而题目只要求返回一组就可以。所以这种方法的虽然可以做到全面搜索，但是较复杂

nums[i] 2 7 4 5
index 0 1 2 3
lookUp 7 2 5 4

换个思路，还是需要一本字典，但这次是边搜索边更新字典,这种方法最多只需n次

Step 0:

序列nums的第0项是2，若要相加为9，则另一项是7。lookUp=9-2=7
在dic中找key为7的项。

key value

若无此项，则将第0项更新到字典后,i++

key value
2 0

Step 1:

序列nums的第1项是7，若要相加为9，则另一项是2。lookUp=9-7=2
由于在dic有key为2的项，则不用继续搜索返回它们对应的index就可以

Code:

let twoSum = function(nums, target) {
    let i
    let lookUp
    let len = nums.length;
    let dic = {};

    for (i=0; i<len; i++) {
        lookUp = target-nums[i];
        if (dic[lookUp] !== undefined){
           return [dic[lookUp],i];   
        }
        dic[nums[i]] = i;
    }
};
