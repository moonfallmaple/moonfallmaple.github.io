Web Crawling

1 基本介绍

网络数据抓取获取分两阶段：抓取和解析

1.1 抓取

抓取的过程就是客户机发送请求（Request）给服务器，服务器返回响应(Response)，拿到响应后再解析

  • python内建模块urllib：已经逐渐被第三方库Request替代
  • 第三方库Request：适合中小型网络爬虫开发
  • Scrapy框架：适合大型网络爬虫开发

1.2 解析

  • BeautifulSoup库
  • re模块

注：另一种方式是由第三方API自动抓取+解析并定期更新
