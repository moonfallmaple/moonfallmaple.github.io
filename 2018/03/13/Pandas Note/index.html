Pandas Note

Pandas主要有两个数据结构：Series和DataFrame

1.Series

它是单一列。DataFrame 中包含一个或多个 Series，每个 Series 均有一个名称。
表现形式为：索引在左边，值在右边。

city = pd.Series(['a','b','c']) #a,b,c是字符要带上‘’
0    a
1    b
2    c
dtype: object

population = pd.Series([20,30,40])
0    20
1    30
2    40
dtype: int64

2.DataFrame

是一个表格型的数据结构，它包含有一组有序的列，每列可以是不同的值类型（数值，字符串，布尔值等）。DataFrame既有行索引也有列索引， 它可以被看做由Series组成的大字典。

pd.DataFrame({'City':city , 'Population':pop})

   City    Population
0    a    20
1    b    30
2    c    40

2.1选择元素

df.loc[行名，列名]
election.loc['Bedford', 'winner']#选择行标签为'Bedford',列标签为'winner'的元素
election.iloc[4, 4] == election.loc['Bedford', 'winner']

2.2选择行或列

名称 代码 备注
Select Single-Row df.loc[‘row’] 单行
Select Multiple-Row df.loc[‘X’:’Y’] 连续多行
Select Multiple-Row df.loc[[‘X’,’Y’,’Z’]] 任意多行
Select Single-Col df[‘col’] 单列
Select Multiple-Col df.loc[: ,’A’:’B’] 连续多列
Select Multiple-Col df.loc[: ,[‘A’,’C’]] 任意多列

这一步相当于sql中的SELECT

2.2.1 行：

#用label选择单行

#用label选择多行
