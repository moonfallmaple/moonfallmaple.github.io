Sklearn(Iris Dataset)

1 基本介绍

本文的数据集是常见的官方数据集鸢尾花iris.csv
数据集：由3类不同的鸢尾花构成，每一类数量为50，共有150个观测值
数据集特征：sepal length（萼片长）, sepal width（萼片宽）, petal length（花瓣长）, petal width（花瓣宽）
数据集标签：setosa，versicolor，virginica

2 导入数据

from sklearn import datasets
iris = datasets.load_iris()
print(iris.data)

4 训练

由于鸢尾花数据集的标签是按类别分的（categorical）所以是分类问题。这里用KNN近邻和逻辑回归两种模型来训练数据，来感受一下这两种模型对于准确率的影响。

4.1 方法一：K-nearest neighbors (KNN)算法

#1.将整个数据集拆分为两部分：训练集和测试集
X = iris.data
y = iris.target
from sklearn.cross_validation import train_test_split
X_train,X_test,y_train,y_test = train_test_split(X,y,test_size=0.4,random_state=4)

#2.配置Knn分类算法：K=5
from sklearn.neighbors import KNeighborsClassifier
Knn = KNeighborsClassifier(n_neighbors = 5)
print(knn)

KNeighborsClassifier(algorithm='auto', leaf_size=30, metric='minkowski',
           metric_params=None, n_jobs=1, n_neighbors=1, p=2,
           weights='uniform')
#3.训练
knn.fit(X_train, y_train)

#4.分类
y_pred = knn.predict(X_test)

#5.评估
from sklearn import metrics
print(metrics.accuracy_score(y_test,y_pred))
返回结果：0.9666666666666667
  • 注意：K的取值会影响分类的结果，如当k=1时，分类的准确率就下降了
    ` bash
    knn = KNeighborsClassifier(n_neighbors=1)

knn.fit(X_train, y_train)

y_pred = knn.predict(X_test)
print(metrics.accuracy_score(y_test,y_pred))
返回结果：0.95

- **那么问题来了k应该取什么值才能让分类能更准确呢**

``` bash
k_range = list(range(1,26))
scores = []
for k in k_range:
    knn = KNeighborsClassifier(n_neighbors=k)
    knn.fit(X_train, y_train)
    y_pred = knn.predict(X_test)
    scores.append(metrics.accuracy_score(y_test, y_pred))
  • k的取值对分类准确率的影响如下图
    ` bash
    import matplotlib.pyplot as plt

plt.plot(k_range, scores)
plt.xlabel(‘Value of K for KNN’)
plt.ylabel(‘Testing Accuracy’)
